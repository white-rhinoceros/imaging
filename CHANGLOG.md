# Changelog

## 0.1.1 (2026-02-08)

### RU
- Исправлено: определение типа изображения по расширению теперь нечувствительно к регистру (например, `JPG`/`jpg`).
- Улучшено: допускается расширение с ведущей точкой (например, `.jpg`).

### EN
- Fixed: image type detection by file extension is now case-insensitive (e.g. `JPG`/`jpg`).
- Improved: accepts extensions with a leading dot (e.g. `.jpg`).
